Detailed elevation maps for identifying geologic features

These maps were created for a presentation to the Pennsylvania Geographical Society given by Dr. Brent Zaprowski. The maps were created using 1/3 arc elevation data downloaded from USGS online and transformed with ArcMap 10.5. The study area is broken down into a grid structure to preserve as much detail as possible.

As a part of Salisbury University Horticulture Department our intern team created and maintained a database for the campus's arboretum. The tree location data was collected over a two year period by various interns using surveying equipment. Then the data was turned into 2d and 3d displays for online use through ArcScence and ArcGIS online.

View Project

Evaluating the positional differences between the Google Maps and the United States Census Bureau geocoding APIs

A publication from our GIS programming course at Salisbury, we investigated discrepancies between known locations and Google Maps and the United States Census Bureau's geocoding APIs. We developed a python script using the addresses of known points to geocode them and create a new layer of the returned coordinates, then found the distances between them to evaluate the accuracies of each geocoding service.

View Project

Sea surface temperature maps describing El Nino and La Nina patterns

To make these maps first historic water temperature data was collected from NOAA.gov, which was subjected to a GDAL quality control script. From there the maps were created using ArcMap 10.4.
